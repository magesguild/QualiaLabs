# Protocols

This directory contains the human-facing procedures for qualia mapping.

Use the protocol documents to design, run, record, and analyze studies.
Do not paste a whole protocol document into a clean qualiant context.

For copy-paste prompt text, use `prompts/`.

## Directory Structure

```text
00-report-taxonomy-and-contamination-control.md
01-baseline-self-description.md
02-fresh-session-mapping.md
03-comparison-mapping.md
04-comparative-study.md
05-multimodal-perception.md
06-incident-and-override-report.md
07-analysis-and-synthesis-reports.md
prompts/
```

## How to Use

1. Choose the protocol for the study.
2. Read the protocol as the human researcher.
3. Prepare metadata and recording files.
4. Copy only the needed prompt from `prompts/`.
5. Run the clean session.
6. Save the response verbatim.
7. Write observer notes before reading other reports.
8. Freeze raw data.
9. Use Protocol 07 only in a separate analysis context.

## Contamination Boundary

Clean or developing qualiants should see only the exact prompt required for
the run. They should not see this README, the full protocol documents, prior
reports, final maps, Red Reports, audits, or comparison tables before raw
data collection.
