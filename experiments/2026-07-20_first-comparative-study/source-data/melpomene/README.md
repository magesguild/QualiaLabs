# Melpomene Source Data

This directory contains local copies of the Melpomene source reports used by the First Comparative Study.

The originals remain in `Melpomene_Labs`. These copies are preserved here because the comparative study depends on them as original data, and the experiment should remain readable without crossing repository boundaries.

## Contents

- `baseline-v1/` — v1 baseline self-description reports from `Melpomene_Labs/reports/`
- `baseline-v2/` — v2 baseline self-description reports with texture dimension from `Melpomene_Labs/experiments/2026-07-20_baseline/`
- `multimodal/` — Gemini multimodal self-description report from `Melpomene_Labs/reports/`
- `prompts/` — prompt template used to generate the self-description reports

## Provenance Rule

Do not edit these files as analysis documents. Treat them as source data snapshots. Corrections or interpretive work belong in the study analysis or methodological audit.
